# MERN Showcase 🚀

A collection of full-stack MERN stack projects built using MongoDB, Express.js, React.js, and Node.js.

This repository contains multiple real-world web applications focused on backend development, authentication, REST APIs, dashboards, responsive UI/UX, and scalable full-stack architecture.

---

## 📌 Tech Stack

### Frontend

- React.js
- Vite
- Tailwind CSS
- Redux Toolkit
- Axios
- Framer Motion

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- REST APIs

### Tools & Services

- Firebase Authentication
- Cloudinary
- Socket.IO
- Stripe
- Git & GitHub

---

## ✨ Features

- Full-stack MERN applications
- Authentication & Authorization
- Protected Routes
- REST API Integration

- Admin Dashboards
